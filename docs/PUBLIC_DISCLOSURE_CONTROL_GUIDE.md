# Public Disclosure Control Guide

## Can Be Shown Publicly

- High-level platform narrative
- Conceptual candidate ranking
- Roadmap CAD/FEM/prototype/validation
- Partner access
- Research positioning

## Must Not Be Shown Publicly

- Formulas
- Exact geometry
- Magnet orientation details
- Dimensions
- CAD parameters
- FEM parameters
- Real ranking algorithm
- Manufacturing-ready drawings

## Approved Language

- Proprietary research framework
- Confidential architecture under development
- IP strategy in progress
- Restricted research access
- Future FEM validation

## Prohibited Language

- Patented
- Patent pending
- Exact claims
- Guaranteed performance
