## Proyecto Principal

**KODEX M-001**

- Imanes permanentes de neodimio
- Configuraciones magnéticas experimentales
- Bobinas tipo Möbius
- Gemelo Digital (Fusion 360)
- Simulación electromagnética
- Validación experimental
