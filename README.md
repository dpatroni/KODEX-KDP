# KODEX Discovery Platform v1.0 Alpha

A premium research-console prototype for KODEX Research Labs.

## Run local

```bash
npm install
npm run dev
```

## Main routes

- `/` Spanish public landing
- `/en` English public landing
- `/dashboard` Research dashboard
- `/explorer` Architecture Explorer
- `/live-lab` Interactive preview
- `/hall-of-fame` Best candidates
- `/experiments`
- `/publications`
- `/roadmap`
- `/collaboration`
